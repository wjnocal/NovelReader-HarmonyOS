# 小说阅读器 HarmonyOS

小说阅读器（NovelReader）的 HarmonyOS 版本，基于 ArkTS 与 ArkUI 开发，并持续同步 Android 版的主要功能与使用体验。

- 当前版本：`1.1.0`（`versionCode 1010000`）
- Bundle Name：`com.wjnocal.novelreader`
- 支持设备：HarmonyOS 手机、平板
- Android 版本：[wjnocal/NovelReader](https://github.com/wjnocal/NovelReader)

## 1.1.0 更新重点

- 新增独立的在线搜书界面，可自由选择书源、输入关键词或地址并查看搜索结果。
- 同名小说按书名和作者合并展示，下载前可选择实际使用的书源。
- 支持将在线小说下载为 TXT 或 EPUB，并可直接加入书架。
- 完善目录分页、章节正文多页合并、失败重试、书源限流和并发下载调度。
- 支持后台下载；切换到其他界面或应用进入后台后，下载任务可继续执行并保留进度。
- 新增内置浏览器与净化在线阅读，可从小说网页直接进入与本地阅读器一致的阅读界面。
- 在线阅读可识别“上一章”“下一章”“上一页”“下一页”等网页导航，章节内存在多页时会先续接正文，再跨章节阅读。
- 在线阅读支持点击中间呼出菜单、点击左右区域或横向滑动翻页；阅读时自动隐藏菜单。
- 修复异形屏安全区、按钮文字截断、分页内容超出屏幕和在线搜书交互动画不同步等问题。
- 更新应用名称、图标和内置书源规则。

## 功能

- 本地导入 TXT、EPUB 小说
- 自动解析中英文常见章节标题，支持 EPUB 2 NCX 与 EPUB 3 Navigation 目录
- 重复导入或重复下载同名同作者书籍时更新内容，并保留书架实体和阅读进度
- 滚动阅读与滑动翻页阅读
- 字号、行距、页边距及阅读主题设置
- 目录、书签、笔记和文本标记
- 书架网格/列表展示、搜索、排序、分类、置顶和改名
- 阅读时长、连续阅读天数及每日阅读目标统计，切入后台时自动保存本次阅读
- 多书源在线搜书、同名结果聚合、来源选择及 TXT/EPUB 下载
- 目录分页、章节正文多页解析、失败重试、限流调度和后台下载
- 内置小说浏览器，支持网址或关键词访问、前进后退、刷新和加载进度，可将当前章节直接切换为净化阅读模式
- 在线阅读支持书源规则匹配、通用正文识别、广告文本过滤、章节内多页续接和跨章节跳转
- 在线阅读与本地阅读保持一致的分页/滚动、手势、菜单、字号、行距、页边距和主题设置
- WebDAV 双向同步书籍、进度、书签、笔记与阅读统计，包含删除同步和并发冲突保护
- 本地数据备份与幂等恢复（同名书合并，书签和笔记去重）
- 手机、平板及横屏布局适配

## 运行项目

1. 使用 DevEco Studio 打开项目目录。
2. 等待工程同步及依赖初始化完成。
3. 在项目结构的“签名配置”中选择或生成自己的调试签名。
4. 连接 HarmonyOS 手机、平板或模拟器。
5. 运行 `entry` 模块。

首次运行后可从主页导入本地 TXT/EPUB，或进入“在线搜书”选择书源搜索和下载小说。

也可从“主页 → 内置浏览器”访问小说站点，在具体章节页点击右下角“在线阅读”进入正文模式。在线模式适合临时阅读；需要长期保存、添加书签或同步阅读数据时，建议通过在线搜书下载到书架。

## WebDAV 同步

在“主页 → WebDAV 同步”中填写服务器地址、用户名和密码，测试连接成功后即可执行双向同步。

同步目录及数据格式与 Android 版本保持兼容，默认使用：

```text
NovelReaderSync/manifest.json
```

建议首次同步前备份本地数据，并避免两台设备同时修改同一本书的阅读数据。

同步清单使用 ETag 条件写入；如果发布前服务器数据已被其他设备更新，本次同步会停止并提示重试，避免静默覆盖。

## 在线搜书

在线搜书内置 25 个与 [wjnocal/NovelReader](https://github.com/wjnocal/NovelReader) 安卓版同步的书源，底层规则参考 [freeok/so-novel](https://github.com/freeok/so-novel)。可在搜索前自行选择书源；来自不同书源的同名同作者小说会合并为一项，点击后再选择具体来源和 TXT/EPUB 下载格式。

当前规则适配 CSS 选择器、动态搜索参数、JSONP 搜索结果、Base64 正文、`data-id` 段落重排及脚本分页。书源仍可能因目标网站结构、Cloudflare、代理要求或网络环境变化而失效。

下载过程会遵守各书源配置中的请求间隔，并支持目录分页、章节多页正文合并、失败重试和安全的并发加速。后台任务使用 HarmonyOS `dataTransfer` 模式；系统省电策略、网络限制或书源限流仍可能影响下载速度。

## 内置浏览器与在线阅读

内置浏览器支持输入完整网址或搜索关键词，并提供后退、前进、刷新和加载进度。打开小说章节网页后，点击“在线阅读”可提取标题与正文，过滤常见广告内容并进入净化阅读界面。

在线阅读器会优先使用匹配书源的正文与导航规则，并在没有匹配规则时使用通用网页识别。它可以识别中文或英文的上一章、下一章、上一页、下一页按钮，以及常见的 `rel`、ID、class 导航标记。若一章被网站拆成多页，会先合并下一页正文，读完后才进入下一章，并避免重复链接造成循环跳转。

分页模式下可以点击屏幕左右区域或横向滑动翻页，点击中间区域呼出/隐藏阅读菜单；滚动模式下也会在阅读时隐藏菜单。字号、行距、页边距和主题设置与本地阅读器保持一致。

应用需要以下网络权限：

```text
ohos.permission.INTERNET
ohos.permission.KEEP_BACKGROUND_RUNNING
```

## 数据说明

书籍解析结果、阅读进度、书签、笔记和设置保存在应用沙箱中。卸载应用可能清除这些数据，请提前使用备份或 WebDAV 同步功能保存。

书架主数据和章节数据采用临时文件替换写入；主数据损坏时会尝试读取最近一次可用备份。
